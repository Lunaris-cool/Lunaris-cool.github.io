Creating Links

To create a proxy or game link, you can use static hosts to deploy. Many proxies may not allow you to deploy statically because they have fullstack backends.

To deploy these proxies go to freedns.afraid.org and make an account.

Choose a domain of your choice from the registry (check if it's unblocked) then type in the BYOD ip of the site.

Fastly

Fastly is requiring you to add a payment method now.

For those of you with legacy fastly accounts, you can add a cdn service:

Click to copy: your-name.global.ssl.fastly.net

Import from literally anywhere. This method can work on school wifi also, and the domain is unblocked for many.

  2. Set DNS servers:
    66.23.198.252

Corkey

  1. Powerwash Chromebook
  2. Turn off Wi-Fi
  3. Restart
  4. Check extensions

Aurora

Recovery image/shim manager with WiFi support. Merged version of Priism & IRS.

  1. Build or download image
  2. Flash & boot
  3. Unenroll with quicksilver
Warning: Improper flashing can brick your Chromebook
